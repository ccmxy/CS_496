Community Map Tutorial

By using this tutorial, you can learn to create an app which features a community map for all users of the app to add their own markers with custom messages to. We will be using Parse.com for the back end and displaying the markers in Google Maps. You do not need to have ever used Parse.com or Google Maps before-- that's what this tutorial is focused on teaching you.

Also, we will be learning to implement ParseObjects in multiple ways so that you can better understand how it works.

This guide does assume that you have some familiarity with Android Studio and have it installed.
